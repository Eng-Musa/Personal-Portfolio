EMS

This project enhances billing accuracy and efficiency for landlords and real estate companies. Key features include landlord management, automatic token reload, remote power control, power usage metrics, and tenant access via mobile app and USSD. The technology stack comprises sensors coded in MicroPython and C++, Java backend, Angular frontend, and databases like PostgreSQL, MySQL, and MongoDB.

E-BEE

This project is designed to help Tatu City residents move around efficiently and economically. It utilizes e-bee electric bikes equipped with GPS and remote control mechanisms. Users can rent these bikes, travel around the city, and settle their payments at the end of their journey. The system is built using Java, Angular, and Kotlin.
