EMS

Developed and deployed an advanced electricity metering system with the following key features:

  • Real-Time Monitoring: Instantly track energy usage.
  • Enhanced Security: Tamper-proof mechanisms, overvoltage protection.
  • User-Friendly Interfaces: Easy-to-use controls and clear displays.
  • Remote Access: Recharge credits and access usage data from anywhere.
  • Advanced Data Analytics: Gain detailed insights into consumption patterns for informed decision-making.
  • Web Access: Available for admin, super admin, landlords, and tenants.
  • Mobile Access: Tenants can use a mobile app and USSD.
  • Technology Stack: DLMS/COSEM protocol, wireless communication, SpringBoot backend, Angular frontend.
  • Databases: PostgreSQL and MongoDB.

Serving 150+ customers testing phase.

BIKE-SHARING

Developed and tested a smart e-bike rental system for Tatu City residents with the following key features:

  • Efficient Transport: Rent e-bee electric bikes for convenient city travel.
  • GPS Tracking: Bikes equipped with GPS for accurate location and tracking.
  • Remote Control Mechanism: Secure bike management with remote control features.
  • Alert System: Notifies if users move outside the Tatu City area.
  • Bike Suggestions: Suggests the nearest available bike based on user location.
  • Payment System: Settle payments at the end of the journey.
  • Technology Stack: Built with Java for backend, Angular for frontend, and Kotlin for mobile apps.
  • Databases: PostgreSQL for data storage.

Projected daily usage: 2000+ users.
